# SQL Server 2025 — procedimentos `sp_` (lista completa em Markdown)

Fonte base: documentação oficial da Microsoft Learn para `System stored procedures (Transact-SQL)` (`view=sql-server-ver17`), consultada em 2026-04-18.

> Escopo deste arquivo: **todos os procedimentos documentados com prefixo `sp_`** no hub oficial de *system stored procedures*.
>
> Ficam fora da lista principal:
> - procedimentos que **não** começam com `sp_`, como `sysmail_*_sp`;
> - procedimentos `xp_*`;
> - famílias documentadas em hubs separados, como alguns conteúdos de replicação.

## Resumo

- Categorias com itens `sp_`: 23
- Ocorrências somadas por categoria: 352
- Procedimentos `sp_` únicos: 320
