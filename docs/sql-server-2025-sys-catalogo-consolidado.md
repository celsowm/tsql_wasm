# Catálogo de Sistema SQL Server (SSMS Compatibility)

Este documento consolida as colunas essenciais que o SSMS 19/20 espera encontrar nas views do schema `sys`.

## sys.objects / sys.all_objects
- name (sysname)
- object_id (int)
- principal_id (int, NULL)
- schema_id (int)
- parent_object_id (int)
- type (char(2))
- type_desc (nvarchar(60))
- create_date (datetime)
- modify_date (datetime)
- is_ms_shipped (bit)
- is_published (bit)
- is_schema_published (bit)

## sys.columns / sys.all_columns
- object_id (int)
- name (sysname)
- column_id (int)
- system_type_id (tinyint)
- user_type_id (int)
- max_length (smallint)
- precision (tinyint)
- scale (tinyint)
- collation_name (sysname, NULL)
- is_nullable (bit)
- is_ansi_padded (bit)
- is_rowguidcol (bit)
- is_identity (bit)
- is_computed (bit)
- is_filestream (bit)
- is_replicated (bit)
- is_non_sql_subscribed (bit)
- is_merge_published (bit)
- is_dts_replicated (bit)
- is_xml_document (bit)
- xml_collection_id (int)
- default_object_id (int)
- rule_object_id (int)
- is_sparse (bit)
- is_column_set (bit)
- generated_always_type (tinyint)
